hasAnyRole
service in module angularShiro.directives

Description

Display if the current Subject is assigned any of the specified roles (for example, 'ADMIN' or ['ADMIN','DEVELOPPER','GUEST'])

Usage

as attribute
<ANY has-any-role="{string |
array | expression}">
   ...
</ANY>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 600.

Parameters

ParamTypeDetails
hasAnyRolestring array expression

the specified role names (for example, 'ADMIN' or ['ADMIN','DEVELOPPER','GUEST'])
